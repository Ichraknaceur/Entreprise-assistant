# Software Access Request Process

## Purpose
This document explains how employees request access to software applications and business systems.

## Request Submission
All requests must be submitted through the access management portal. Requests must specify the application, business justification, and requested level of access.

## Approval Flow
Manager approval is required for standard access. Sensitive or privileged access also requires application owner approval and, in some cases, security review.

## Provisioning Time
Standard access requests are typically processed within two business days. Privileged access may take longer depending on controls.

## Access Reviews
Access rights are reviewed periodically to ensure they remain appropriate for the employee’s role.

## Removal of Access
Access must be removed when an employee changes role, leaves the company, or no longer requires the system.
